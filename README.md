## Project Overview
HealthInOne is a Django web application that allows users to log daily health metrics
(water, sleep, exercise, calories) and receive a daily score and personalised recommendations.

## Features
- User accounts and profiles
- Daily health logging
- Automated scoring engine
- Personalised recommendation engine
- Dashboard with history chart
- View previous days
- Leaderboard based on 7-day average

## How to Run

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
